
# Dockey

![Dockey Logo](app/frontend/public/logo.png)

Dockey is a microservices-based document management platform. It provides document editing, user management, comments, and more, with a web interface and robust backend services.

## Services
- **docs-service**: Document management
- **user-service**: User authentication and profiles
- **comments-service**: Comments and likes
- **checker**: Content moderation
- **frontend**: Web interface (React)
- **keycloak**: Authentication provider
- **prometheus**: Monitoring
